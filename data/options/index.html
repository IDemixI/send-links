API Key

For the extension to share links and clipboard content with other computers on your network, you need to get your free and private API Key from websocket.in and insert the API Key in all computers that you would like to be able to send or receive links. If you are using another WebSocket server (like your own private WSS server) that does not require an API Key, still fill the box with an arbitrary string.

-

  1. This is the name of your browser which is seen by all members of your network. It is recommended to name the browser such that it reveals both the computer and browser names (e.g.: chrome-desktop)
  2. This is a Secure WebSocket server that is used to connect all the computers on your network together. You can use the default server or create your own. You can even have a local WSS server so that the extension could send links and share clipboard to the other computers on your network even if the INTERNET is not connected!
  3. If a password is present, the data that is sent to or received from the WebSocket server is encrypted end to end. This way the WebSocket server cannot read messages between peers. Note that you will need to enter the same password on all machines and then restart each extension for all the peers to be able to read the encrypted messages.
